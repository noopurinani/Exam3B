About the Final Project

The following flow diagrams depict how the entire model facilitates the functionality of the web application created for our final project. As the purpose of this project was to create (add), update, delete and read tasks and users for both admins and users, each flow diagram depicts how various servlets connect to query classes and the database. These flow diagrams were vital in understanding how each aspect of our code was connected to ensure that data was not only added and updated, but also read and even deleted.
