About Homework 3

Homework 3's purpose is to set up the NetBeans development environment, create a dynamic web page in JSP and publish that page on a TomCat Server.

This particular web page is a two page web application that enables employees to estimate their net pay for the week

The assumptions include a tax rate of 18% for those with gross pay less than $500 each week, 22% for those with $500 or more a week in gross pay and that overtime is 1.5 times the hourly rate for any hours worked over 40 hours a week.
